# Inceptyon Labs Homebrew Tap

Homebrew formulae and casks for Inceptyon Labs projects.

## Installation

```bash
brew tap inceptyon-labs/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `freeport` | Your turnstile for AI coding assistants |
| `mise-app` | Grocery lists, recipes, and meal planning |

### Freeport

```bash
brew install --cask freeport
```

A macOS app that unifies Claude, Gemini, Copilot, and other AI coding assistants into a single endpoint with quota tracking and automatic failover.

[Learn more](https://github.com/inceptyon-labs/freeport)

### Mise

```bash
brew install --cask mise-app
```

A macOS app for grocery lists, recipes, and meal planning — everything in its place. Syncs across devices via iCloud.

[Learn more](https://github.com/inceptyon-labs/mise)
